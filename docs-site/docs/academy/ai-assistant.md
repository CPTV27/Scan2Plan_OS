# AI Assistant

Chat-based AI help for any platform question.

## Accessing the Assistant

Click the **Chat** icon in the sidebar, or use the Chat tab in FieldHub.

## What It Can Do

- Answer questions about the platform
- Explain how features work
- Help troubleshoot issues
- Suggest next steps for leads
- Draft email responses
- Explain pricing calculations

## Context Awareness

The assistant knows:
- Current page/context
- Your role and permissions
- Active project data
- Recent actions

## Example Prompts

- "How do I build a quote?"
- "Why is my travel cost showing $0?"
- "What disciplines should I select for a hospital?"
- "Help me draft a follow-up email for this lead"
