# AI Features

Scan2Plan OS leverages AI throughout the platform.

## Available AI Features

| Feature | Description |
|---------|-------------|
| **Intelligent Scoping** | Suggests building type, LOD, disciplines from project description |
| **Document Intelligence** | Extracts requirements from RFPs/PDFs |
| **Predictive Deal Intelligence** | Win probability and risk analysis |
| **Natural Language CPQ** | Build quotes from plain English |
| **AI Proposal Generator** | Personalized proposals by buyer persona |
| **Smart Project Matching** | Find similar past projects using vector search |
| **Voice Transcription** | Whisper AI for voice notes |
| **Field Note Translation** | Convert raw notes to professional scope |

## AI Models Used

- **GPT-4o-mini**: Text generation and analysis
- **Gemini Pro**: Embeddings and structured data
- **Whisper**: Voice transcription

## Usage Limits

AI endpoints are rate-limited:
- 10 requests per minute for AI endpoints
- Usage logged for cost tracking
