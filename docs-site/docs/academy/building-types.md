# Building Types

Scan2Plan OS supports 16 building types, each with its own complexity factor.

## Available Types

| Building Type | Complexity |
|---------------|------------|
| Commercial - Simple | Low |
| Commercial - Complex | High |
| Office | Medium |
| Hospital | Very High |
| Industrial | Medium |
| Warehouse | Low |
| Retail | Medium |
| Educational | Medium-High |
| Residential - Single Family | Low |
| Residential - Multi-Family | Medium |
| Historical/Renovation | Very High |
| Religious Building | High |
| Hotel/Resort | High |
| Theatre/Performing Arts | High |
| Museum/Gallery | High |
| Built Landscape | Per-acre pricing |

## Complexity Impact

Higher complexity buildings have:
- More detailed requirements
- Longer scanning time
- More intricate BIM modeling
- Higher per-sqft pricing
