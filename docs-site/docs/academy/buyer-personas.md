# Buyer Personas

Buyer Personas help tailor communication based on your prospect's role and priorities.

## Available Personas

| Code | Role | Communication Focus |
|------|------|---------------------|
| **BP1** | Engineer | Technical detail, risk, coordination |
| **BP2** | Executive | ROI, speed, bottom line |
| **BP3** | Project Manager | Timeline, budget, deliverables |
| **BP4** | Facilities Manager | Operations, maintenance, documentation |
| **BP5** | Architect | Design accuracy, precision, aesthetics |
| **BP6** | Owner/Developer | Value, investment return, asset protection |
| **BP7** | GC/Contractor | Schedule, coordination, change orders |

## Using Personas

1. Set the Buyer Persona on a lead
2. Proposal Builder uses persona to tailor language
3. AI Assistant adjusts recommendations

## Persona in Proposals

The Proposal Builder automatically:
- Adjusts headline language
- Emphasizes relevant benefits
- Includes persona-specific case studies
- Uses appropriate technical depth
