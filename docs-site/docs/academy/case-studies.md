# Case Studies

Add relevant case studies to proposals to build credibility.

## Adding Case Studies

1. In the Proposal Builder, find the Case Studies section
2. Click **"Add Case Study"**
3. Select from the Evidence Vault
4. System recommends matches by building type

## Evidence Vault Integration

Case studies come from the Evidence Vault:
- Project photos
- Success metrics
- Client testimonials
- Before/after comparisons

## Auto-Recommendations

The system suggests case studies matching:
- Building type
- Disciplines
- Project size
- Industry vertical
