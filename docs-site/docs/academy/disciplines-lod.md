# Disciplines & LOD

Configure which building systems to model and at what level of detail.

## Disciplines

| Discipline | What's Included |
|------------|-----------------|
| **Architecture (Arch)** | Walls, floors, ceilings, doors, windows |
| **Structural (Struct)** | Beams, columns, foundations, slabs |
| **Mechanical (Mech)** | HVAC, ductwork, equipment |
| **Electrical (Elec)** | Conduit, panels, fixtures, cable trays |
| **Plumbing (Plumb)** | Pipes, fixtures, valves |
| **Site** | Exterior, topography, hardscape |

## Level of Detail (LOD)

| LOD | Detail Level | Use Case |
|-----|--------------|----------|
| **LOD 200** | Basic geometry, placeholders | Early design, conceptual |
| **LOD 300** | Precise geometry | Standard deliveries |
| **LOD 350** | High detail with connections | Coordination |
| **LOD 400** | Fabrication-ready | Shop drawings, prefab |

## Pricing Impact

Higher LOD = more modeling hours = higher cost.

Each discipline can have its own LOD selection.
