# Job Lifecycle

Every project in Scan2Plan follows a defined lifecycle from lead to delivery.

## Stages

### 1. Lead
A potential project enters the pipeline with basic information:
- Client/Company name
- Project name and address
- Initial contact details

### 2. Quote (CPQ)
Configure the project scope and pricing:
- Select building type
- Define square footage
- Choose disciplines (Arch, Struct, MEP)
- Set LOD levels
- Add risk factors
- Calculate travel costs

### 3. Proposal
Generate a professional proposal document:
- Section-based templates
- Dynamic variable substitution
- Case study integration
- Brand-compliant formatting

### 4. Closed Won
Client signs the proposal:
- Lead converts to Project automatically
- Quote data transfers to project
- Production scheduling begins

### 5. Production
Work execution phases:
- **Scheduling**: Assign field tech, set scan date
- **Scanning**: On-site data capture
- **Processing**: Point cloud registration
- **Modeling**: BIM model creation
- **QC**: Quality control check

### 6. Delivery
Final handoff:
- Export final deliverables
- Send to client
- Archive project
