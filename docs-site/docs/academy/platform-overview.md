# Platform Overview

Scan2Plan OS is a comprehensive operating system for laser scanning and BIM services companies. It integrates all aspects of your business into one cohesive platform.

## Core Modules

### Sales Pipeline
Track leads from first contact through closed deals. Visual Kanban board with drag-and-drop stage management.

### CPQ (Configure-Price-Quote)
Build accurate quotes with:
- 16 building types with complexity factors
- Multi-discipline support (Arch, Struct, MEP, Site)
- LOD levels (200, 300, 350, 400)
- Risk factor adjustments
- Automated travel pricing

### Proposal Builder
Generate professional proposals with:
- Template-based sections
- Dynamic variable substitution
- Case study integration
- PandaDoc/DocuSeal e-signature

### Production Tracker
Manage active projects through:
- Scheduling → Scanning → Processing → Modeling → QC → Delivery

### FieldHub Mobile
Mobile-optimized interface for field technicians:
- GPS time tracking
- Voice notes with AI transcription
- Photo/video capture
- Quick actions

### Financial Module
Track profitability with:
- Profit First accounting allocations
- QuickBooks integration
- Invoice management
- Job costing analytics
