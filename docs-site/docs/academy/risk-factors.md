# Risk Factors

Risk factors add percentage markup to account for project complexity and challenges.

## Available Risk Factors

| Factor | Markup | Description |
|--------|--------|-------------|
| **Occupied Building** | +15% | Work around occupants |
| **Hazardous Conditions** | +25% | Safety requirements |
| **No Power/HVAC** | +20% | Battery equipment needed |
| **Remote Location** | +10% | Extended travel |
| **Fast Track/Rush** | +15% | Compressed timeline |
| **Revisions Expected** | +10% | Scope changes likely |
| **Incomplete Documents** | +10% | Discovery work required |
| **Difficult Access** | +15% | Limited entry points |
| **Multi-Phase** | +10% | Multiple mobilizations |
| **Union Site** | +5% | Union labor rules |
| **Security Requirements** | +5% | Badging, escorts |

## Applying Risk Factors

1. In the Quote Builder, expand "Risk Factors"
2. Check applicable factors
3. Total risk multiplier applies to base price

## Custom Risk Factors

CEO role can configure custom risk factors in **Settings > CPQ Settings**.
