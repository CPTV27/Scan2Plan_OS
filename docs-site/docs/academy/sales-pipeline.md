# Sales Pipeline

The Sales Pipeline is your command center for managing leads and opportunities.

## Pipeline Stages

1. **Leads** - New opportunities
2. **Contacted** - Initial outreach made
3. **Proposal** - Quote/proposal sent
4. **Negotiation** - Active discussions
5. **On Hold** - Temporarily paused
6. **Closed Won** - Deal signed ✅
7. **Closed Lost** - Deal lost ❌

## Creating a Lead

1. Navigate to **Sales Pipeline** from the sidebar
2. Click **"+ New Deal"** button
3. Fill in required fields:
   - Client / Company name
   - Project Name
   - Project Address (critical for travel calculations)
4. Click **Save**

## Lead Details

Each lead contains:
- **Client Information**: Company, contact name, email, phone
- **Project Details**: Address, square footage, building type
- **Deal Information**: Value, probability, source
- **Activity Timeline**: All interactions logged

## Moving Deals

Drag and drop deals between stages, or use the deal dropdown menu.

:::warning Lead Source Attribution
You must select a **Lead Source** before moving a deal to "Closed Won".
:::

## Win Probability

Probability is calculated based on:
- Manual estimate (0-100%)
- Stage progression
- Staleness decay (if configured)
