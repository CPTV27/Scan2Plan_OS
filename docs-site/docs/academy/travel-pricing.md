# Travel Pricing

Travel costs are calculated based on distance from dispatch location.

## Dispatch Locations

- **Brooklyn** (primary)
- **Woodstock** (Hudson Valley)
- **Troy** (Capital Region)

## Brooklyn Pricing Tiers

| Tier | Project Size | Base Fee | Per-Mile |
|------|--------------|----------|----------|
| **Tier A** | 50,000+ sqft | $0 | $4/mile over 20 |
| **Tier B** | 10,000-49,999 sqft | $300 | $4/mile over 20 |
| **Tier C** | Under 10,000 sqft | $150 | $4/mile over 20 |

## Woodstock/Troy Pricing

Flat rate: **$3 per mile** (no base fee, no minimum)

## Calculating Travel

1. In Quote Builder, select **Dispatch Location**
2. Enter the **Project Address**
3. Click **Calculate Distance**
4. System uses Google Maps API for accurate mileage
5. Travel cost appears in the pricing preview

:::tip Tier A Auto-Flagging
Projects 50,000+ sqft are automatically flagged as "Tier A" with amber badge and priority 5.
:::
