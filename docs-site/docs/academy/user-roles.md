# User Roles

Scan2Plan OS has role-based access control. Each role has specific permissions.

## Available Roles

### CEO
- Full system access
- Can override margin gates (acknowledge < 40% margin)
- Manage team members and permissions
- Access all analytics and reporting
- Configure business defaults

### Sales
- Sales Pipeline management
- Lead creation and editing
- Quote building (CPQ)
- Proposal creation
- Deal advancement through stages
- Win/Loss analytics

### Production
- Project management
- Field operations (FieldHub)
- Scheduling and assignments
- QC workflows
- Delivery tracking

### Accounting
- Financial module access
- QuickBooks integration
- Invoice management
- Profit First allocations
- Job costing reports

## Role Assignment

Roles are assigned by the CEO role in **Settings > Team**.
