# Voice Notes

Record spoken notes that AI transcribes automatically.

## Recording a Voice Note

1. Tap **Voice Note** Quick Action
2. Press **Record Voice Note** button
3. Speak clearly into microphone
4. Tap **Stop** when finished
5. AI transcribes using OpenAI Whisper
6. Text appears in Field Notes

## Transcription Features

- Automatic punctuation
- Speaker-aware formatting
- Technical term recognition
- Noise filtering

## Saving Notes

Notes save locally until Daily Report submission:
1. Notes remain editable
2. Can re-record sections
3. Attach to specific project
4. Submit with end-of-day report

## Troubleshooting

**Microphone not working?**
- Grant microphone permission in browser
- Ensure HTTPS connection
- Try different browser
