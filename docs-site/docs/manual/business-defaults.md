# Business Defaults

Configure system-wide defaults.

## Location

**Settings > Business Defaults**

## Available Settings

| Setting | Description |
|---------|-------------|
| Default LOD | Standard level of detail |
| Target Margin | Profitability target |
| Payment Terms | Standard payment options |
| Valid Days | Quote expiration default |
| Dispatch Location | Default origin |

## How Defaults Apply

Defaults populate new quotes and leads:
- Can be overridden per-quote
- Save time on data entry
- Ensure consistency
