# Google Workspace

Connect Google Calendar, Gmail, and Drive.

## Available Integrations

- **Calendar**: Schedule field appointments
- **Gmail**: Track email communications
- **Drive**: Store project files

## Setup

1. Go to **Settings > Integrations**
2. Click **Connect Google**
3. Select scopes (Calendar, Gmail, Drive)
4. Authorize access

## Calendar Features

- Create events from scheduling view
- Sync tech availability
- Block time for scans

## Drive Features

- Upload project files
- Link deliverables
- Share with clients
