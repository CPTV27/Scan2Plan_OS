# Invoicing

Create and track invoices for projects.

## Invoice Schedule

| Milestone | Invoice Type | Amount |
|-----------|--------------|--------|
| Scanning starts | Deposit | 50% |
| Delivery | Final | 50% |

## Creating Invoices

1. Navigate to project
2. Click **Create Invoice**
3. Select invoice type
4. Push to QuickBooks (if connected)

## Invoice Gate

Projects cannot be marked "Delivered" until all invoices are paid.

## Tracking

View invoice status:
- Draft
- Sent
- Viewed
- Paid
- Overdue
