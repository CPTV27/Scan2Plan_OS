# Lead Sources

Track where leads come from for attribution.

## Location

**Settings > Lead Sources**

## Managing Sources

- Add new sources (Referral, Website, Trade Show)
- Remove unused sources
- View attribution reports

## Lead Source Gate

:::warning Required for Closed Won
You must select a Lead Source before moving a deal to "Closed Won". This ensures attribution is always tracked.
:::

## Attribution Reporting

View lead source breakdown:
- Total leads by source
- Conversion rates by source
- Revenue by source
