# Profit First Accounting

Revenue allocation following Profit First methodology.

## Target Allocations

| Account | Target % |
|---------|----------|
| Operating Expenses | 30% |
| Taxes | 15% |
| Profit | 10% |
| Owner Pay | 45% |

## Configuration

**Settings > Financial > Profit First**

Adjust percentages to match your business goals.

## How It Works

1. Revenue enters the system via project payments
2. System calculates allocation per account
3. View allocation dashboard
4. Transfer funds on allocation day
