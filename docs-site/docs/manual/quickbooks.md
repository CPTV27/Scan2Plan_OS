# QuickBooks Integration

Connect Scan2Plan OS to QuickBooks Online for invoicing and expense tracking.

## Connecting

1. Go to **Settings > Integrations**
2. Click **Connect QuickBooks**
3. Authorize the OAuth connection
4. Tokens are stored securely

## Features

- Create invoices from projects
- Sync expenses to projects
- Track actual vs. quoted margin
- Push estimates

## Invoice Workflow

1. Project reaches milestone (scanning, delivery)
2. Click **Create Invoice**
3. Invoice created in QuickBooks
4. Link stored in Scan2Plan

## Troubleshooting

**Connection expired?**
Go to Settings > Integrations > QuickBooks and click Reconnect.
