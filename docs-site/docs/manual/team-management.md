# Team Management

Manage users and permissions (CEO only).

## Location

**Settings > Team**

## Inviting Users

1. Click **Invite User**
2. Enter email address
3. Select role
4. Send invitation

## Roles

| Role | Permissions |
|------|-------------|
| CEO | Full access |
| Sales | Pipeline, quotes, proposals |
| Production | Projects, field ops |
| Accounting | Financial module |

## Removing Users

1. Find user in team list
2. Click remove/deactivate
3. Confirm action

User data is preserved for audit trail.
