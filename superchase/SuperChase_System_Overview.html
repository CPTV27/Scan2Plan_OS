How It Works

2

Orchestrator Routes

Determines category, priority, and which agent(s) handle it.

3

4

Low-risk runs auto. High-risk waits for approval.

Every 10 min, Asana ‚Üî Sheets sync both directions.

Troubleshooting

Sync seems delayed?

Changes not showing in Asana or Sheets

Wait 10 min for auto-sync, or run syncAllSheetsToAsana() manually in Apps Script

Task created in wrong project?

Item appeared in SC: Tasks instead of SC: Leads

Move it in Asana - next sync will update the Sheet

Claude doesn't know context?

New session doesn't remember SuperChase

Upload Master Context doc or paste the quick prompt at session start

Voice not working in car?

Audio routing to wrong device

Disconnect glasses OR car - use only one Bluetooth device, then try Gemini Live
